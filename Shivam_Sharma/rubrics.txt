Rubrics

Total: 40 points. Each item is worth 1 point and is scored independently.

1. The game opens directly from index.html. (1 point)
2. No build step or package installation is required to run. (1 point)
3. The game runs offline with no internet connection at runtime. (1 point)
4. The canvas is exactly 960 by 640 pixels. (1 point)
5. The field is a 24 by 16 grid of 40 by 40 pixel tiles. (1 point)
6. A single continuous enemy path is drawn from entry to exit. (1 point)
7. The path bends at least three times. (1 point)
8. The path enters at the left edge tile (0, 8). (1 point)
9. The path exits at the right edge tile (23, 6). (1 point)
10. Path tiles cannot be built on. (1 point)
11. Non path grass tiles can be built on. (1 point)
12. The background uses hex 1A1208. (1 point)
13. The path tiles use hex 6B4A2B. (1 point)
14. The grass tiles use hex 2E4A24. (1 point)
15. The accent and turret fire use hex FF7A1A. (1 point)
16. The player starts with exactly 20 lives. (1 point)
17. The player starts with exactly 100 gold. (1 point)
18. Current lives are shown on screen at all times. (1 point)
19. Current gold is shown on screen at all times. (1 point)
20. The current wave is shown as Wave X / 8 during play. (1 point)
21. Exactly eight waves are implemented. (1 point)
22. Later waves contain more enemies than the first wave. (1 point)
23. At least one faster or tougher enemy type appears after wave 1. (1 point)
24. Enemies spawn one at a time with a short delay between them. (1 point)
25. Each enemy moves along the defined path from entry to exit. (1 point)
26. Each enemy shows a health bar that shrinks as it takes damage. (1 point)
27. An enemy that reaches the exit reduces lives by one. (1 point)
28. Destroying an enemy adds its gold bounty to the player's gold. (1 point)
29. At least two turret types are available to place. (1 point)
30. The turret types differ in cost, range, fire rate, and damage. (1 point)
31. A turret can be placed only on a buildable grass tile. (1 point)
32. Placement is blocked when the player does not have enough gold. (1 point)
33. The turret cost is deducted from gold when it is placed. (1 point)
34. Each turret automatically targets the nearest enemy in range. (1 point)
35. Each turret fires at a fixed rate and removes its damage from enemy health. (1 point)
36. A placed turret can be upgraded by paying an upgrade cost. (1 point)
37. An upgrade increases that turret's damage or range. (1 point)
38. A Start screen with a working Start button begins the game. (1 point)
39. A Win screen appears after wave 8 is cleared with at least one life. (1 point)
40. A Lose screen appears at zero lives and offers restart without a page reload. (1 point)
